## 🎯 핵심 기능

### 1. 프로젝트 완성도 계산
```typescript
function calculateCompletionRate(project): number {
  const fields = [
    project.problem_statement,
    project.target_audience,
    project.situation,
    project.evidence1,
    project.hypothesis1,
    project.solution,
    project.features,
    project.experiment_log,
    project.wrong_assumption,
  ];
  
  const filled = fields.filter(f => f && f.trim() !== '').length;
  return Math.round((filled / fields.length) * 100);
}
```

### 2. 섹션 상태 표시
- ✓ **완료된 섹션**: 흰색 배경 + 체크 마크
- ⚪ **미완료 섹션**: 점선 테두리 + 투명 배경

### 3. 네비게이션 하이라이트
- 현재 페이지: 파란색 배경
- 다른 페이지: 회색 텍스트 → 호버시 배경 표시

## 📱 반응형 디자인

### 데스크톱
- 네비게이션: 로고 | 메뉴 | 사용자
- 프로젝트 섹션: 4열 그리드
- 통계/액션: 2열 그리드

### 모바일 (768px 이하)
- 네비게이션: 로고 | 로그아웃만
- 메뉴 숨김 (햄버거 메뉴는 구현 안 함)
- 프로젝트 섹션: 1열
- 버튼: 전체 너비

## 🧪 테스트 방법

### 1. 네비게이션 테스트
```bash
# 1) 로그인 후 대시보드
http://localhost:3001/

# 2) 각 메뉴 클릭
# - "Help" → /helps
# - "Insight" → /insights
# - "대시보드" → /

# 3) 현재 페이지 하이라이트 확인
# → 파란색 배경 표시됨

# 4) 로그아웃 클릭
# → 홈으로 리다이렉트
```

### 2. 대시보드 테스트
```bash
# 1) 로그인
POST /api/auth/signin

# 2) 대시보드 접속
http://localhost:3001/

# 3) 확인 사항
# ✓ 내 프로젝트 카드 표시
# ✓ 완성도 진행 바 (0-100%)
# ✓ 섹션 상태 (완료/미완료)
# ✓ Help/Insight 통계 (0개)
# ✓ 빠른 액션 링크

# 4) "프로젝트 편집" 클릭
# → /teams/{team_id}/edit로 이동

# 5) Help/Insight 작성 후 대시보드 재방문
# → 통계 숫자 증가 확인
```

## 🎨 디자인 특징

### 프로젝트 카드
- **그라데이션 배경**: 파란색 → 보라색
- **진행 바**: 흰색 바 + 투명 배경
- **섹션 상태**: 
  - 완료: `rgba(255,255,255,0.25)` + 실선
  - 미완료: `rgba(255,255,255,0.1)` + 점선

### 통계 카드
- 호버 시: `translateY(-4px)` + 테두리 표시
- 아이콘: 3xl 크기
- 숫자: 파란색, 3xl, 굵게
